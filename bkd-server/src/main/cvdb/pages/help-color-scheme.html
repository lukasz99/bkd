Help: Varaint and Aminoacid Color Schemes

Common Color Codes for Amino Acids
Group Amino Acids Color Description
Nonpolar Ala(A), Val(V), Leu(L), Ile(I), Met(M) Green Hydrophobic side chains
Aromatic Phe(F), Tyr(Y), Trp(W) Purple Aromatic side chains
Polar, uncharged Ser(S), Thr(T), Asn(N), Gln(Q) Gray Hydrophilic, uncharged side chains
Acidic Asp(D), Glu(E) Red Negatively charged at physiological pH
Basic Lys(K), Arg(R), His(H) Blue Positively charged at physiological pH
Special cases Cys(C), Gly(G), Pro(P) Yellow (Cys), Cyan (Gly,Pro) Distinctive roles in structure
