Lollipop Panel Help

The lollipops show the variants reported in the database selected.

  • If different amino acid changes are reported at the same position, they will be included in the same lollipop.
  • Height of the lollipop/the number displayed reflect the number of variants at that position.
Click on a lollipop to see details about variants reported at a specific position in the specified database.
  • It is possible to select more than one lollipop at the same time.
  • This will display the different amino acid changes reported at this position, with links to ClinGen, ClinVar, dbSNP, and gnomAD if available
  • If there is a link in the CVDB column, that means this variant also contains an entry in the CVDB database.
  • Click on the lollipop again to de-select.

Zoom in to a specific region of the protein using the grey slider bars at the bottom of the graph.
